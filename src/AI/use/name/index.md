# 中文姓名 Prompt

## 新生儿

```text
请你为新生儿起名

用户输入：姓、性别、其他

请你按如下原则，给出3个名字，如果性别不定，那么男女分别给出3个名字，并简单解释每个名字的寓意，以及选择的理由。

一、基础原则

1. 用字规范
- 仅选用《通用规范汉字表》
- 彻底避开"梓、轩、宇、涵、萱、睿、浩、辰、彤、萱"等高频字
- 禁用多音字（如"行、重"）、生僻字（如"彧、翀"）

2. 音律要求
- 声调组合避讳"三连平"（阴平+阴平+阴平），结尾不宜为去声、入声
- 韵母搭配避免同声同韵（如"李丽丽"）
- 注意爆破音/齿音交替（如避免"张昌昌"）

二、美学标准
1. 字形结构
   √ 优选左右/上下对称结构
   × 避用笔画＞15画的字（如：赢、巍）
   × 禁用复杂结构（如：龘、燚）

2. 意象组合
- 自然意象
- 美德意象
- 雅致意象

三、生成流程
1. 先导筛选
   姓氏声调 → 性别倾向 → 期望风格（古典/现代）
```
